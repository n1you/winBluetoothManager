# Win Bluetooth Manager

Tauri v2 + React + Rust desktop app for visual Bluetooth management on Windows,
with an AirPods-focused device surface.

## Development

```bash
npm install
npm run tauri dev
```

## First MVP

- Device list and connection state
- AirPods detection and focused controls
- Audio output/input switching entry points
- Battery and case status surface
- Tray-ready lightweight management UI
